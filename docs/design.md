# UI 设计

本文档介绍 bm.md 的用户界面设计，仅关注视觉与交互层面。

---

## 整体布局

### 页面结构

```
┌─────────────────────────────────────────────────────────────┐
│  [文件1] [文件2] [文件3] [+]              文件标签栏          │
├─────────────────────────┬───────────────────────────────────┤
│                         │                                   │
│                         │                                   │
│       Markdown          │         实时预览                   │
│        编辑器            │                                   │
│                         │      ┌─────────────────┐          │
│                         │      │   设备模拟框    │          │
│                         │      │   (iPhone/     │          │
│                         │      │    Safari)     │          │
│                         │      └─────────────────┘          │
│                         │                                   │
│                         │                                   │
├─────────────────────────┴───────────────────────────────────┤
│  [编辑器操作]           bm.md           [预览器操作]         │
│                        底部功能栏                            │
└─────────────────────────────────────────────────────────────┘
```

### 布局特点

- **文件标签栏** - 顶部显示打开的文件，支持多文件切换
- **左右分屏** - 编辑器与预览器各占 50%
- **可调节分割** - 拖拽中间分割线调整比例
- **最小宽度** - 每侧最小 512px，整体最小 1280px
- **固定底栏** - 高度 48px，承载所有操作按钮

### 文件标签栏

```
┌─────────────────────────────────────────────────────────────┐
│ [📄 bm.md ✕] [📄 README.md ✕] [📄 notes.md ✕]    │ [+] │
│              打开的文件标签                      │ 新建 │
└─────────────────────────────────────────────────────────────┘
```

- **活动标签** - 高亮显示当前编辑的文件
- **关闭文件** - 点击 × 或聚焦标签后按 Delete
- **重命名文件** - 双击标签或聚焦后按 F2
- **滚动支持** - 标签过多时可横向滚动
- **新建按钮** - 右侧固定的 + 按钮创建新文件
- **加载门禁** - 文件正文加载期间显示占位状态，不挂载可编辑区域或使用旧预览执行导出

### 底部功能栏

底栏按左、中、右三组排列：

- **左侧 EditorActionBar** - 导入、导出 Markdown、分隔线、格式化、命令面板入口、设置
- **中间** - Logo（点击打开关于页面）
- **右侧 PreviewerActionBar** - 复制微信、复制 HTML、导出菜单、分隔线、排版样式、代码主题、Mermaid 主题、Infographic 设置、自定义 CSS

## 配色系统

### Ayu 主题

基于 [Ayu](https://github.com/ayu-theme/ayu-colors) 配色方案，统一整个应用的视觉风格：

#### 浅色模式 (Ayu Light)

| 用途      | 颜色名 | 色值      |
| --------- | ------ | --------- |
| 背景      | 雪白   | `#fcfcfc` |
| 次级背景  | 浅灰   | `#f8f9fa` |
| 前景文字  | 石墨灰 | `#5c6166` |
| 强调色    | 橙色   | `#f29718` |
| 链接/信息 | 天蓝   | `#55b4d4` |
| 成功      | 绿色   | `#6cbf43` |
| 错误      | 红色   | `#e65050` |

#### 深色模式 (Ayu Mirage)

| 用途      | 颜色名 | 色值      |
| --------- | ------ | --------- |
| 背景      | 深蓝黑 | `#1f2430` |
| 次级背景  | 靛灰   | `#242936` |
| 前景文字  | 暖灰   | `#cccac2` |
| 强调色    | 金黄   | `#ffcc66` |
| 链接/信息 | 亮青   | `#5ccfe6` |
| 成功      | 浅绿   | `#87d96c` |
| 错误      | 珊瑚红 | `#ff6666` |

### 色彩技术

- **OKLCH 色彩空间** - 使用现代 CSS 色彩格式，确保感知一致性
- **CSS 变量** - 通过 CSS 变量映射到 shadcn/ui 组件
- **语义化命名** - `--primary`、`--secondary`、`--destructive` 等

## 编辑器设计

### CodeMirror 编辑器

````
┌─────────────────────────────────────┐
│  1 │ # 标题                         │
│  2 │                                │
│  3 │ 这是一段正文，包含 **粗体** 文字  │
│  4 │                                │
│  5 │ ```javascript                  │
│  6 │ console.log('Hello')           │
│  7 │ ```                            │
│    │                                │
│    │                                │
└─────────────────────────────────────┘
````

#### 视觉特点

- **行号** - 左侧显示行号，使用次级文字颜色
- **语法高亮** - Markdown 语法实时着色
- **当前行** - 当前编辑行轻微高亮
- **光标** - 使用强调色（橙色）

#### 字体

- **编辑器字体** - 使用 CodeMirror 的默认等宽排版
- **字号与行高** - 由 CodeMirror 与浏览器样式控制，项目不额外固定

## 预览器设计

### 设备模拟框

预览内容展示在设备模拟框中，增强真实感：

#### 移动端视图 - iPhone

```
        ┌─────────────────┐
        │      ━━━ ●      │  ← 灵动岛
        ├─────────────────┤
        │                 │
        │   Markdown      │
        │   预览内容       │
        │                 │
        │                 │
        │                 │
        └─────────────────┘
```

- 外壳宽度：415px，由 375px 内容宽度与左右各 20px 边框组成
- 高度：根据预览区可用空间在 650px 至 850px 之间适配
- 预览 store 的移动模式宽度值：415
- 圆角边框
- 顶部灵动岛装饰（Dynamic Island）

#### 桌面端视图 - Safari

```
┌─────────────────────────────────────┐
│ ● ● ●  │    bm.md    │             │  ← 标题栏
├─────────────────────────────────────┤
│                                     │
│         Markdown 预览内容            │
│                                     │
│                                     │
└─────────────────────────────────────┘
```

- 当前预览使用 `mode="simple"`，最大宽度为 768px
- 标题栏左侧保留三个中性窗口控制点，地址栏居中显示 "bm.md"
- 省略默认模式中的侧栏、前进后退、分享、新建标签、标签页与刷新等额外工具

### 侧边栏

预览区域右侧使用 Lucide 图标按钮与 Tooltip，按从上到下的分组排列：

- **外观与预览** - 应用主题切换（Sun / Moon）、移动端预览（Smartphone）、桌面端预览（Monitor）、预览区独立浅深配色（LampDesk）
- **内部导航** - MCP 配置、Skill 技能
- **外部导航** - API 文档（BookOpen）、Twitter、GitHub、请喝咖啡（Coffee），在新标签页打开

自定义 CSS 和“更多设置”不在侧栏中。

### 自定义 CSS 对话框

点击底部功能栏右侧的画笔按钮打开自定义 CSS 配置：

│  │ /* 修改标题颜色 */                                      │  │
│  │ #bm-md h1 {                                            │  │
│  │   color: #e74c3c;                                      │  │
│  │ }                                                      │  │
│  │                                                        │  │

│  │ #bm-md p {                                             │  │
│  │   line-height: 1.8;                                    │  │
│  │ }                                                      │  │
│  └───────────────────────────────────────────────────────┘  │
│                                                             │

- **实时预览** - 修改后即时应用到预览区域

- **语法提示** - 提示选择器约束规则

## 加载状态

编辑器与预览器共用紧凑的 loading fallback：在 `bg-editor` 容器中央显示单行 `text-xs` 辅助文字。fallback 同时覆盖客户端挂载和懒加载阶段，不使用全屏遮罩、骨架屏或装饰动画。

## 响应式设计

### 预览模式

预览区域提供两种由用户明确选择并持久化的模式：

| 预览模式 | 设计宽度 | 外观                      |
| -------- | -------- | ------------------------- |
| 移动端   | 415px    | iPhone 设备框（外壳宽度） |
| 桌面端   | 768px    | Safari 浏览器框           |

拖动编辑器与预览区之间的分割线只改变可用空间，不会切换预览模式或重建预览 iframe。

### 最小屏幕要求

- 最小宽度：1280px（5xl）
- 最小高度：700px

### 容器查询

使用现代 CSS 容器查询实现父元素感知的响应式布局：

- `@container` 规则用于组件内部适配
- 支持 `orientation="responsive"` 的表单字段布局

## 组件风格

### 按钮

- **主要按钮** - 强调色背景，白色文字
- **次要按钮** - 透明背景，边框样式
- **图标按钮** - 仅图标，悬停显示 Tooltip
- **圆角** - 统一使用 `rounded-none`（直角风格）

### 下拉菜单

- 直角卡片样式
- 轻微阴影
- 选中项显示勾选标记

### 输入框

- 直角边框
- 聚焦时边框高亮
- 清晰的占位符文字

### 滚动条

- 细线样式（6px 宽）
- 仅悬停时显示
- 与主题配色一致
